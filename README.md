# Blog Management System

A full-stack **Blog Management System** built using **React.js** and **ASP.NET Core Web API**.

This application allows users to create, view, update, and delete blog posts with a responsive user interface and REST API integration.

---

## Features

- Create Blog Posts
- View Blog Posts
- Update Existing Blogs
- Delete Blogs
- Responsive User Interface
- REST API Integration
- Smooth Animations and Transitions
- API-based Data Communication

---

# Project Architecture

## Frontend

The React.js frontend communicates with backend APIs using Axios to perform CRUD operations.

## Backend

The ASP.NET Core Web API handles:

- API endpoints
- Business logic
- Database operations
- Data access using Entity Framework Core

# Installation & Setup

## Clone Repository

```bash
git clone https://github.com/Ashfana3012/BlogManagementSystem-Ashfana.git
```

---

# Frontend Setup

Navigate to the frontend folder:

```bash
cd Client
```

Install dependencies:

```bash
npm install
```

Run the React application:

```bash
npm run dev
```

The frontend will run on:

```
http://localhost:5173
```

---

# Backend Setup

1. Open the ASP.NET Core Web API project in Visual Studio.

2. Configure the required settings in:

```
appsettings.json
```

3. Run the API project.

The backend API will start on the configured localhost port.

---

# API Integration

The React application communicates with ASP.NET Core Web API using Axios.

Implemented operations:

- GET - Retrieve blog posts
- POST - Create blog posts
- PUT - Update blog posts
- DELETE - Remove blog posts
